# Hackathon Galáctica: WDK Edition 1

## Introduction

**Hackathon Galáctica: WDK Edition 1** is the first hackathon in Tether's Hackathon Galáctica series—an online program designed to explore agents as economic infrastructure and accelerate the next generation of agentic builders.

From **25.02.2026 to 22.03.2026**, developers will build AI agents and applications that hold wallets, move money, and settle value onchain—powered by self-custodial infrastructure across 6+ chains.

AI agents are no longer just tools — but they are also not social experiments. In this hackathon, we explore **agents as economic infrastructure**: autonomous systems that execute tasks, manage capital, and interact with onchain logic under clearly defined constraints.

> **Builders define the rules → Agents do the work → Value settles onchain**

Rather than optimizing for discourse, engagement, or popularity, projects are evaluated on **correctness, autonomy, and real-world viability**.

*While others chase hype, we build what lasts.*

Please read the **Rules** and **Terms of Participation** carefully before deciding if you want to participate. You are required to agree to both the Rules and Terms of Participation in order to participate in this Hackathon. The Rules and Terms of Participation should be read in conjunction with the Privacy Statement for the Hackathon.

### Track Details

#### 🤖 Agent Wallets (WDK / OpenClaw and Agents Integration)

Build agents or bots that integrate directly with Tether's Wallet Development Kit (WDK), enabling native USD₮, USA₮ and XAU₮ wallets, custody, and on-chain transactions. This track focuses on turning agents into real financial actors.

**Must Have:**
- OpenClaw (or any other equivalent agent framework) for agent reasoning
- Use WDK primitives directly (wallet creation, signing, accounts)
- Agents should hold, send, or manage USD₮, USA₮ or XAU₮ autonomously

**Nice to Have:**
- Clear separation between agent logic (OpenClaw) and wallet execution (WDK)
- Emphasis on safety: permissions, limits, recovery, or role separation
- Smart accounts or account abstraction (optional)

**Bonus:**
- Composability with other agents or protocols
- Use open-source LLM frameworks

**🏆 Total Prize Pool: 5,000 USD₮** — 🥇 1st: 3,000 USD₮ · 🥈 2nd: 2,000 USD₮

#### 💰 Lending Bot

How do trust and risk evolve when agents control capital? Build autonomous agents that issue loans, assess risk, and manage debt without human intervention. Develop credit-scoring systems, undercollateralized lending, or P2P agent markets to reimagine finance for a world where AI agents are independent economic actors.

**Must Have:**
- The agent makes lending decisions without human prompts
- All transactions settle on-chain using USD₮, USA₮, XAU₮ or BTC
- The agent autonomously tracks and collects repayments

**Nice to Have:**
- Use DIDs or on-chain history for agent credit scores
- Use LLMs to negotiate loan terms with borrowers
- The agent reallocates capital to higher-yield opportunities
- Mechanics allow for lending with minimal or no collateral

**Bonus:**
- Agents borrow from other agents to complete complex tasks
- Agents use their own earned revenue to service debt
- Machine learning models predict the probability of default
- Zero-Knowledge Proofs verify credit without exposing private data

**🏆 Total Prize Pool: 5,000 USD₮** — 🥇 1st: 3,000 USD₮ · 🥈 2nd: 2,000 USD₮

#### 🌊 Autonomous DeFi Agent

Create agents that interact with DeFi protocols to deploy capital, manage liquidity, optimize yields, or rebalance positions. This track emphasizes programmable finance and agents that reason about opportunity and risk.

**Must Have:**
- Agent decides **when and why** (not just how)
- USD₮, USA₮ or XAU₮ used as base asset, quote asset, or settlement layer
- WDK for wallet and transaction execution

**Nice to Have:**
- DeFi protocols (DEXs, lending, vaults)
- Awareness of slippage, risk, or protocol state
- Clear abstraction: strategy → execution
- OpenClaw for strategy and planning

**Bonus:**
- Composability with other agents or protocols
- Use open-source LLM frameworks

**🏆 Total Prize Pool: 5,000 USD₮** — 🥇 1st: 3,000 USD₮ · 🥈 2nd: 2,000 USD₮

#### 💸 Tipping Bot

Build agent-powered tipping experiences on top of **Rumble**, a video platform that has integrated Tether's Wallet Development Kit (WDK) as a native wallet for creator tipping. Extend Rumble's tipping flows with autonomous agents, programmable logic, and smarter creator–fan interactions powered by supporter tokens (USD₮, XAU₮, and BTC).

**Must Have:**
- Build on top of Rumble's existing USD₮, USA₮, XAU₮, or BTC tipping wallet, not a standalone payments app
- Use agents to enhance tipping: automation, personalization, or conditional logic

**Nice to Have:**
- Creator monetization, fan engagement, and real-time interactions
- Keep UX simple and instant—tipping should feel native to video consumption
- OpenClaw for agent logic and orchestration

**Example ideas:**
- Agent that auto-tips creators based on watch time or engagement
- Community-driven tipping pools managed by an agent
- Smart splits between creators, collaborators, or causes
- Event-triggered tipping (livestream moments, milestones, reactions)

## Judging Criteria

Projects are evaluated based on:

1. **Technical correctness** — Sound architecture, clean integrations, and reliable execution. Proper use of relevant tools (e.g. WDK, OpenClaw, protocols) and working end-to-end flows are expected.
2. **Degree of agent autonomy** — How independently the agent operates. Strong submissions demonstrate planning, decision-making, and execution without constant human input, rather than simple scripts or manual triggers.
3. **Economic soundness** — Sensible use of USD₮, XAU₮, or other assets, with attention to incentives, risk, and sustainability. The agent's behavior should make economic sense in real conditions.
4. **Real-world applicability** — Clear user value, realistic use cases, and a plausible path to adoption. Projects should feel deployable beyond the hackathon, not just proof-of-concept demos.
